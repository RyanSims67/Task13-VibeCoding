# Task B — Lovable Prompts

This file contains the prompts I used in Lovable for Task B.

The goal was to create a middle-sized pet project called CampusClubHub.

## Prompt 1 — Start from Task A Design

I have a Google Stitch GUI design for an app called CampusClubHub.

Build a middle-sized student club management app based on this idea.

CampusClubHub should help a student club manage events, members, announcements and event sign-ups.

The app should include:

* dashboard
* upcoming events
* member list
* announcements
* create event form
* event sign-up status
* simple navigation

Make it look like a modern but simple club management app.

## Prompt 2 — Make It Middle-Sized

Expand CampusClubHub into a middle-sized pet project.

Add separate sections for:

* Dashboard
* Events
* Members
* Announcements
* Calendar

Keep the app simple enough for a student project, but make it feel bigger than a basic ToDo list.

## Prompt 3 — Improve the Dashboard

Improve the dashboard layout.

Show cards for:

* upcoming events
* total members
* pending sign-ups
* latest announcements

Make the design clear and easy to explain in documentation.

## Prompt 4 — Add Realistic Club Features

Add realistic student club features.

Include:

* create event button
* event date and location
* member roles
* announcement cards
* sign-up count for each event
* calendar preview

Keep the design beginner-friendly and not too complex.

## Prompt 5 — Final Simplification

Simplify the final version so I can explain it easily.

Keep the most important features:

* dashboard
* events
* members
* announcements
* calendar
* event sign-ups

The app should feel like a middle-sized pet project, not a huge professional system.

## Result

Lovable generated a larger CampusClubHub prototype with multiple sections.

I used the result as my Task B evidence for a vibe-coded middle-sized pet project.
