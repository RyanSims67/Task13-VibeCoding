# Task B — CampusClubHub Feature List

This file lists the main features planned or generated for CampusClubHub in Lovable.

## Feature 1 — Dashboard

The dashboard gives a quick overview of the club.

It can show:

* upcoming events
* total members
* latest announcements
* pending sign-ups

This makes the app feel more complete than a single-page project.

## Feature 2 — Events

The events section shows club events.

Each event can have:

* title
* date
* location
* short description
* number of sign-ups

This is one of the main parts of the app.

## Feature 3 — Create Event

The app includes the idea of creating a new event.

A club organizer could enter:

* event name
* date
* location
* description

## Feature 4 — Members

The members section shows people in the club.

A member can have:

* name
* role
* status
* activity information

This makes the app feel more like a club management system.

## Feature 5 — Announcements

The announcements section is for club updates.

Example announcements could be:

* meeting reminders
* event changes
* general club news

## Feature 6 — Calendar

The calendar section gives an overview of upcoming club activities.

This helps users see what is happening soon.

## Feature 7 — Event Sign-Ups

The app can show how many people signed up for an event.

This is useful for club planning.

## Why These Features Fit Task B

Task B asks for a bigger or middle-sized pet project.

CampusClubHub fits this because it has several connected parts:

* dashboard
* events
* members
* announcements
* calendar
* sign-ups

It is larger than a basic ToDo list but still understandable as a student project.
