# Task A — Google Stitch Prompts

This file contains the prompts I used in Google Stitch for Task A.

The goal was to create a GUI design for CampusClubHub before continuing the same idea in Lovable for Task B.

## Prompt 1 — First GUI Design

Create a clean GUI design for a student club management app called CampusClubHub.

The app should help a student club manage events, members, announcements and event sign-ups.

Include:

* app title
* dashboard overview
* upcoming events section
* member list preview
* announcements section
* create event button
* event sign-up status
* simple navigation sidebar or bottom navigation

Make it look like a modern but simple student club app.

## Prompt 2 — Make It Suitable for a Middle-Sized Pet Project

Expand the GUI idea so it feels like a middle-sized pet project.

The app should have clear sections for:

* Dashboard
* Events
* Members
* Announcements
* Calendar

Keep the layout simple enough that I can explain it in my documentation.

## Prompt 3 — Make It Easier to Continue in Lovable

Adjust the design so I can continue or transfer the idea into Lovable.

The design should clearly show:

* dashboard cards
* event list
* member overview
* announcement area
* create event form idea
* navigation

Avoid overly complex animations or advanced features.

## Prompt 4 — Final Refinement

Create a final beginner-friendly GUI layout for CampusClubHub.

The design should look like a realistic student club dashboard.

Focus on the main workflow:

* view club overview
* see upcoming events
* manage members
* post announcements
* create events
* view sign-up status

## Result

The final Google Stitch design will be used as the starting visual idea for Task B in Lovable.
