## Task A

For Task A, I used **Google Stitch** to create a GUI design for CampusClubHub.

CampusClubHub is a student club management app idea with events, members, announcements and sign-ups.

The Google Stitch design was used as the visual starting point for Task B.

## Task B

For Task B, I used **Lovable** to create a middle-sized pet project called CampusClubHub.

The project is an app for a student club with:

* events
* members
* announcements
* sign-ups
* calendar view
* dashboard overview

CampusClubHub Lovable Website:

```text
https://campusclubhub.lovable.app
```

## Task C

For Task C, I built a separate distributed Python app called CampusMessenger using **Google Antigravity IDE**.

CampusMessenger is a simple client/server messenger app.

It includes:

* Tkinter GUI client
* API client
* Flask server
* message routes
* message service
* JSON message storage
* shared message model
* tests

The goal for Task C was to create a modular app and understand every code file.
